# Security Policy

## Supported versions

This project is a portfolio MVP currently maintained for educational purposes.

| Version | Supported |
| ------- | --------- |
| MVP     | Yes       |

## Reporting a vulnerability

If you find a security issue, please do not open a public issue.

Please report it privately through GitHub's private vulnerability reporting feature when available.

Include:
- a clear description of the issue
- steps to reproduce it
- the affected part of the application
- possible impact

We will review the report and take appropriate action.
