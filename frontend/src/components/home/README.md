# Home Components

## Purpose
This folder contains landing page sections and visuals for the public homepage.

## Analogy
Think of this folder as the product front desk: it introduces the marketplace and guides visitors toward registration or login.

## Contents
- `HomeHero.jsx` renders the main landing hero section.
- `HeroImageShowcase.jsx` renders the hero visual composition.
- `HomeBenefits.jsx` renders the benefits section.
- `HomeHowItWorks.jsx` renders the process section.
- `HomeProductPreview.jsx` renders product preview content.
- `FeaturedSuppliersCarousel.jsx` renders featured supplier content.
- `HomePricing.jsx` renders pricing content.
- `HomeFinalCta.jsx` renders the final call-to-action section.
- `PublicFooter.jsx` renders the public footer.
- `SectionHeading.jsx` provides reusable section heading structure.
- `LandingIcon.jsx` and `ArrowIcon.jsx` provide landing-specific icons.

## How It Fits in KERNO
These components build the public landing page and communicate the product value before users enter authenticated workflows.

## Maintenance Notes
Keep landing content reusable and data-driven where practical. Avoid moving dashboard or authenticated app behavior into this folder.
