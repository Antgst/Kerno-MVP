# Catalog Pages

## Purpose
This folder contains the route-level product catalog screen.

## Analogy
Think of this folder as the marketplace aisle: it lets stores browse and compare available product offers.

## Contents
- `CatalogPage.jsx` renders the catalog page and coordinates product discovery UI.

## How It Fits in KERNO
The catalog page connects product service data with catalog components so stores can search, filter, and open product details.

## Maintenance Notes
Keep catalog presentation components in `components/catalog/`. Page-level logic should stay focused on route state and data coordination.
