## MVP Scope

KERNO remains a B2B marketplace MVP focused on supplier profiles, store profiles, supplier products, catalog/search, and contact or quote requests. It does not include payment, cart, delivery, or a full order workflow.

## Maintenance Notes

Update these documents when authentication, authorization, token handling, request ownership checks, profile ownership checks, security smoke tests, or sensitive configuration behavior changes.

KERNO reste un MVP de marketplace B2B centré sur les profils fournisseurs, les profils magasins, les produits fournisseurs, le catalogue/la recherche et les demandes de contact ou de devis. Il n'inclut pas le paiement, le panier, la livraison, ni un workflow de commande complet.
