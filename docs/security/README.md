# Security Docs

## Purpose
This folder documents security-related decisions for authentication and API access.

## Analogy
Think of this folder as the access policy reference: it explains how the product protects authenticated workflows.

## Contents
- `AUTH_SECURITY_NOTES.md` documents authentication security notes.

## How It Fits in KERNO
Security documentation supports backend auth, JWT handling, protected routes, and role-based access for supplier and store workflows.

## Maintenance Notes
Update this folder when authentication, authorization, token handling, or sensitive configuration behavior changes.
