# Architecture Docs

## Purpose
This folder documents the structure and technical organization of the KERNO application.

## Analogy
Think of this folder as the system map: it shows how the major parts of the product connect before you inspect individual files.

## Contents
- `APPLICATION_ARCHITECTURE.md` describes the high-level frontend, backend, and database architecture.
- `BACKEND_STRUCTURE.md` documents the modular monolith backend structure.
- `FRONTEND_STRUCTURE.md` documents the React frontend structure and route organization.
- `FRONTEND_CSS_ARCHITECTURE.md` documents frontend CSS organization.

## How It Fits in KERNO
These documents provide the context behind the folder-level README files and help contributors understand where new work belongs.

## Maintenance Notes
Update these documents when architecture or ownership boundaries change, not for every small implementation detail.
